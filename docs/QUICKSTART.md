# 快速启动指南

## 第一次使用

### 1. 环境准备

```bash
# 进入项目目录
cd "D:\Programs\Python\qq邮箱作业收发"

# 激活虚拟环境
venv\Scripts\activate

# 验证依赖
pip list
```

### 2. 配置验证

检查 `.env` 文件配置是否正确：

```env
QQ_EMAIL=你的QQ邮箱
QQ_PASSWORD=QQ邮箱授权码（不是登录密码！）
TARGET_FOLDER=目标文件夹名
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
API_KEY=你的阿里云API密钥
LLM_MODEL=正确的模型名称
```

**重要提示：**
- QQ_PASSWORD 需要在 QQ 邮箱设置中生成授权码
- LLM_MODEL 可能需要更改为：`qwen-plus`、`qwen-turbo` 或 `qwen-max`

### 3. 初始化数据库

```bash
python -m database.schema
```

### 4. 运行测试

```bash
python test_system.py
```

预期输出：
```
============================================================
QQ邮箱作业收发AI系统 - 系统测试
============================================================
[OK] 配置验证通过
[OK] 创建提交记录成功
[OK] 文件存储成功
```

### 5. 启动应用

```bash
python main.py
```

### GUI 操作说明

#### 筛选功能
- **搜索框**: 输入学号或姓名进行搜索
- **学生下拉**: 选择特定学生查看其所有作业
- **作业下拉**: 选择特定作业查看所有学生
- **状态下拉**: 筛选正常/逾期提交

#### 批量操作
1. 在列表中勾选要操作的记录
2. 点击相应的批量操作按钮：
   - **批量下载**: 下载选中作业的附件
   - **批量回复**: 发送确认邮件给学生
   - **批量删除**: 删除本地记录和/或邮箱邮件
   - **导出Excel**: 导出选中记录到 Excel 文件

#### 统计信息
左侧面板显示：
- 总提交数
- 已下载数
- 已回复数

## 常见问题

### Q: AI 提取失败怎么办？
A: 系统会自动使用正则表达式 fallback 模式，仍然可以提取学生信息。

### Q: 如何获取 QQ 邮箱授权码？
A:
1. 登录 QQ 邮箱网页版
2. 进入"设置" -> "账户"
3. 开启"POP3/SMTP服务"
4. 生成授权码（16位字符）

### Q: 邮件监听如何工作？
A:
- 当前版本：手动处理收件箱中的未读邮件
- 后续版本：将支持后台自动监听

### Q: 如何设置作业截止时间？
A: 当前版本需要在数据库中手动设置，后续将添加 GUI 界面。

### Q: 重复提交如何处理？
A: 系统会自动保留最新版本，删除旧版本的本地文件，并更新数据库记录。

### Q: 如何备份数据？
A:
- **数据库**: 复制 `assignment_submissions.db` 文件
- **作业文件**: 复制 `submissions/` 目录

## 数据文件位置

- **配置文件**: `.env`

## 性能优化建议

1. **大量邮件处理**: 分批处理，每次不超过 100 封
2. **数据库查询**: 已添加索引，性能良好
3. **文件存储**: 建议定期清理旧文件

## 安全建议

1. **定期备份数据库和作业文件**
2. **不要将 .env 文件提交到版本控制**
3. **定期更改 QQ 邮箱授权码**
4. **保护 API Key 不泄露**

## 技术支持

如遇问题，请检查：
1. Python 版本（建议 3.8+）
2. 虚拟环境是否激活
3. 依赖包是否完整安装
4. .env 配置是否正确
5. 网络连接是否正常
