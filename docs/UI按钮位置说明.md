### 详细位置

  1. 批量下载
  2. 批量回复
  3. 批量删除
  4. 导出Excel

## 如何找到按钮

### 步骤1：启动应用
```bash
python main.py
```

### 步骤2：查看左侧面板
应用启动后，左侧面板包含：
- 筛选区域（搜索、学生、作业、状态）
- 统计信息
- **批量操作区域** ← 在这里

### 步骤3：定位批量操作
向下滚动左侧面板，找到：
1. "统计信息"标题
2. 其下方就是"批量操作"区域

### 步骤4：确认按钮
在"批量操作"标题下，应该看到：
- "已选择: 0 项"（计数器）
- 4个按钮垂直排列

## 可能看不到的原因

### 1. 窗口太小
**解决方法：**
- 最大化窗口
- 或手动调整窗口大小（至少900像素高）

### 2. 需要滚动
**解决方法：**
- 在左侧面板向下滚动
- 找到"批量操作"区域

### 3. 面板被遮挡
**解决方法：**
- 确保左侧面板完全可见
- 检查是否有其他窗口覆盖

### 4. UI未正确加载
**验证方法：**
- 检查控制台是否有错误信息
- 查看是否显示"启动图形界面..."

## 视觉特征

**批量操作区域特征：**
- 大号粗体标题："批量操作"
- 橙色/棕色背景的框架（CTkFrame）
- 4个蓝色按钮（customtkinter默认蓝色）
- 按钮文本：白色

**计数器特征：**
- 显示："已选择: 0 项"
- 数字会根据选择变化
